how to connect to a VPN

1. On the far right of the taskbar, select the Network icon

2. Select the VPN connection you want to use, then do either of the following depending on what happens when you select the VPN connection: -If the Connect button displays under the VPN connection, select Connect. -If the VPN section in Settings opens, select the VPN connection there, then select Connect.

3. If you're prompted, enter your username and password or other sign-in info.

When connected, the VPN connection name will display Connected underneath it. To see if you’re connected to the VPN while you’re doing things on your PC, select the Network icon (either or ) on the far right of the taskbar, then see if the VPN connection says Connected.
